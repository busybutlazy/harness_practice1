# AGENTS.md

## 專案目標
建立一個最小化的全端 Todo List 示範專案，用於學習 Codex harness 模式。

## 技術棧
- 前端：React + Vite
- 後端：FastAPI
- API 基礎路徑：`/api`

## 規則
- 請以Docker專案建制方式。
- 當任務同時涉及前端與後端時，請啟動子代理（subagents）。
- 在建立或修改這個示範專案之前，先執行 `$scaffold-fullstack`。
- 若需要任何 OpenAI／Codex／MCP 的設定細節，請優先使用 OpenAI 開發者文件的 MCP 伺服器。
- 保持實作精簡。除非有明確要求，否則不要加入驗證、資料庫或樣式函式庫。
## 完成條件
- 前端可以從後端列出任務。
- 前端可以新增任務。
- 前端可以切換任務的完成狀態。
- 前端能關注任務，並將關注任務釘選在最上方。
- 後端可成功執行。
- 前端可成功執行。
- 回傳一段簡短摘要，說明已變更的檔案，以及接下來需要手動執行的命令。
